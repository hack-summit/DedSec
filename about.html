VULNERABILITY SCANNER
ABOUT VULNERABILITY SEVERITY CONTACT

Developers code everyday, not every developer codes safely. Due to this, the applications
developed are insecure. A developer knows to fix a bug, but where is the endpoint to fix? Many
companies do not have security engineers, leading to a high risk of data breach. So, we need to
create a scanner which finds security bugs with more accuracy and which is customisable
according to the application. We will a build application that will help both red team and blue
team to automate their security testing.
